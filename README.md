# Simulador de Elevador
[![Java](https://img.shields.io/badge/Java-17%2B-orange)](https://www.java.com)
[![NetBeans](https://img.shields.io/badge/NetBeans-IDE-blue)](https://netbeans.apache.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Projeto acadêmico desenvolvido para simular o funcionamento básico de um elevador, com interface gráfica intuitiva. O sistema permite controlar a lotação máxima, direção (subindo/descendo) e movimentação de pessoas.

## 📋 Funcionalidades

- **Controle de lotação**: Capacidade máxima de 10 pessoas
- **Indicador de direção**: Mostra se o elevador está subindo ou descendo
- **Interface gráfica**: Botões para todas as operações do elevador
- **Validações**: Impede operações que excedam a capacidade ou façam sentido incorreto
